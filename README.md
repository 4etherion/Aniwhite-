# 🌸 Flower - Discover the Beauty of Flowers

A beautiful, modern website showcasing the elegance and beauty of flowers, featuring an AI-powered chat assistant.

## ✨ Features

- **Elegant Design**: Soft pink gradient themes on a clean white background
- **Responsive Layout**: Perfect viewing experience on all devices
- **Flower Gallery**: Beautiful showcase of various flower types
- **AI Chat Assistant**: Interactive chat powered by advanced AI technology
- **Google Authentication**: Secure sign-in for chat features
- **Markdown Support**: Rich text formatting in chat messages
- **Code Highlighting**: Beautiful code snippets with copy functionality

## 🚀 Quick Start

1. **Install Dependencies**
```bash
npm install
```

2. **Run the Server**
```bash
npm start
```

3. **Open in Browser**
Navigate to `http://localhost:3000`

## 🎨 Design Features

- **Color Palette**: White background with soft pink accents (#ffb6c1, #ffc0cb, #ff69b4)
- **Custom Logo**: Three-petal flower design with gradient effect
- **Smooth Animations**: Floating petals and shiny text effects
- **Modern UI/UX**: Clean, intuitive interface with beautiful transitions

## 💬 AI Chat Features

- **Flower-1 Max Model**: Advanced AI assistant for flower-related queries
- **Instant Responses**: Quick, contextual answers
- **Markdown Rendering**: Support for formatted text, lists, and code blocks
- **Copy Functionality**: Easy copying of messages and code snippets
- **Mobile Optimized**: Full-screen chat on mobile devices
