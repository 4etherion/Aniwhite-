# Flower - Discover the Beauty of Nature 🌸

A beautiful, modern website showcasing the elegance and beauty of flowers with an integrated AI chat assistant.

## Features ✨

### Design
- **Clean & Elegant**: White background with soft light pink accents
- **Responsive**: Perfect on desktop, tablet, and mobile devices
- **Beautiful Typography**: Playfair Display and Poppins fonts
- **Smooth Animations**: Delightful scroll animations and transitions

### Content
- **Hero Section**: Welcoming introduction to the beauty of flowers
- **Gallery**: Stunning collection featuring:
  - Roses 🌹
  - Tulips 🌷
  - Cherry Blossoms 🌸
  - Lilies 🪷
  - Peonies 💐
  - Orchids 🌺
- **About Section**: Information about the website's mission
- **Contact Section**: Creator information and social links

### AI Chat Assistant 🤖

**Flower-1 Max** - An intelligent AI companion with:

- **Google Authentication**: Secure sign-in required
- **Markdown Support**: Rich text formatting (headers, bold, italic, code, etc.)
- **Code Highlighting**: Separate copy functionality for code blocks
- **Smart Responses**: Instant, context-aware replies
- **Thinking Animation**: Beautiful shiny text effect while processing
- **Copy Feature**: Easy copying of AI responses
- **Fullscreen Mode**: (Desktop) Click to expand, click overlay to close
- **Mobile Optimized**: Fixed bottom-right corner button

#### Chat Capabilities:
- Detailed flower and plant information
- Website navigation and information
- General knowledge on various topics
- Code examples and technical help
- Friendly, conversational interaction

## Technology Stack 💻

- **HTML5**: Semantic structure
- **CSS3**: Modern styling with gradients, animations, and flexbox/grid
- **JavaScript ES6+**: Interactive features and AI chat
- **Marked.js**: Markdown parsing for chat messages
- **Google Fonts**: Playfair Display & Poppins
- **SVG**: Custom logo and icons

## Getting Started 🚀

### Local Development

1. Clone or download the repository
2. Open `index.html` in a modern web browser
3. No build process required - pure HTML/CSS/JS!

### File Structure

```
flower/
├── index.html          # Main HTML structure
├── styles.css          # All styling and animations
├── script.js           # JavaScript functionality and AI chat
└── README.md          # This file
```

## AI Chat Usage 🗨️

1. **Click** the pink chat button in the bottom-right corner
2. **Sign in** with Google (demo mode for development)
3. **Start chatting** with Flower-1 Max!

### Example Questions:
- "Tell me about roses"
- "What flowers are featured on this website?"
- "Show me a code example"
- "What is the symbolism of cherry blossoms?"

### Markdown Examples in Chat:

The AI supports full markdown:
- **Bold**: `**text**`
- *Italic*: `*text*`
- `Code`: `` `code` ``
- Headers: `## Heading`
- Code blocks: ` ```language ... ``` `

## Customization 🎨

### Colors (CSS Variables)

Edit in `styles.css`:

```css
:root {
    --primary-pink: #ffb6c1;
    --light-pink: #ffc0cb;
    --soft-pink: #ffe4e9;
    --pale-pink: #fff0f3;
    --dark-pink: #ff69b4;
}
```

### AI Responses

Modify the response functions in `script.js`:
- `getFlowerResponse()`
- `getWebsiteResponse()`
- `getCodeResponse()`
- etc.
